Segmented Donut Chart Animation (SVG, pure SCSS, no JS)

For a job application in early 2018, it was my task to animate a donut chart graphic. The choice of implementation was completely up to me. I am a big fan of finding CSS-only approaches to frontend-patterns. So here is my pure CSS, JavaScript-free magical SVG donut chart.

Normally I wouldn't have the animation repeat, but for the sake of showcasing, I find it convenient this way.

Of course, Internet Explorer and Edge don't like CSS transforms with SVG, but for the readability and reuse of code I refrained from adding SVG animations.

Currently responsiveness is due to a second SVG with modified viewport and locations for everything. I could've modified the existing SVG via JS, but wanted to keep it pure.

Another caveat of this approach is that the animation starts instantly. I could add a delay, however this would be random instead of starting when the object comes into view. To achieve this, JS can be used.
